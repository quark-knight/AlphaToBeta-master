 # AlphaToBeta: A deep reinforcemnet learning model for predicitng amino-acid mutations that induce α-helix → β-sheet transitions in proteins
 This is the code repository for the AlphaToBeta Project at Prof. Arnab Mukherjee's CCB lab at IISER Pune.

 **The code is still under being changed and the model is in traning phase**

The question we want to answer is that can we have a prdictive model that can predict point amino acid mutations, that leads to α-helix → β-sheet transitions in proteins.

We are using ESMFold with joint-embedding space of both the helix sequence and environmental sequence with the objective to maximize the reward which basically mean getting heigher α-helix than β-sheet in secondary structure determination.
